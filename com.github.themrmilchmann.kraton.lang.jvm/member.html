fun IJvmType.member(className: String, vararg typeParameters: IJvmType, nullable: Boolean = false): JvmMemberType (source)

Creates and returns a new IJvmType which holds a reference to a member of the receiver type.

Parameters

className - the name of the member to be referenced

typeParameters - the type-parameters for the type

nullable - whether or not the referenced type is nullable

Receiver
the type that contains the member to create a reference to

Return
the newly created IJvmType
